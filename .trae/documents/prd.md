## 1. 产品概述

话费优惠券省钱雷达是一款面向中国手机用户的智能话费优惠聚合H5应用，帮助用户快速发现、精准匹配并一键领取各大运营商的话费充值优惠券和促销活动，实现话费支出的最优化。
- 解决用户在分散渠道中难以找到有效话费优惠的痛点，提供一站式精准优惠信息聚合服务
- 目标用户为全国超过16亿手机用户，通过精准的运营商匹配和智能省钱计算器，帮助用户每月节省10%-50%的话费支出

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览优惠、使用计算器、查看攻略 |
| 注册用户 | 手机号注册 | 收藏优惠、订阅提醒、个性化推荐 |

### 2.2 功能模块
1. **雷达首页**: AI扫描动画、今日最佳优惠、实时更新时间戳、运营商快捷筛选
2. **全部优惠券页**: 按运营商分类的完整优惠券列表、筛选与搜索、优惠券详情与领取攻略
3. **省钱计算器页**: 输入话费金额、选择运营商、智能匹配最优优惠组合、展示省钱策略
4. **我的页面**: 用户信息、收藏优惠、省钱记录、AI对比说明

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：活力橙红 (#FF6B35) + 深邃藏蓝 (#1A1A2E)，辅以翡翠绿 (#00D68F) 作为省钱/成功指示色
- 按钮风格：圆角胶囊按钮，主按钮带微渐变和阴影，次要按钮为描边风格
- 字体：标题使用 DIN Alternate 风格的数字展示，正文使用系统字体
- 布局风格：卡片式布局，底部固定导航栏，顶部沉浸式状态栏
- 图标风格：Lucide线性图标，搭配运营商品牌色点缀

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 雷达首页 | AI扫描动画区 | 深色背景，脉冲圆环动画，荧光绿扫描线，白色文字时间戳 |
| 雷达首页 | 今日最佳优惠 | 渐变橙红背景卡片，大号金额数字，闪光标签"今日最佳" |
| 雷达首页 | 热门优惠速览 | 横向滚动卡片，圆角阴影，运营商色标 |
| 雷达首页 | 运营商快捷入口 | 三个圆形图标按钮，品牌色背景 |
| 全部优惠券页 | 筛选Tab | 固定顶部Tab栏，选中态带下划线动画 |
| 全部优惠券页 | 优惠券卡片 | 白色圆角卡片，左侧运营商色条，右侧"去领取"按钮 |
| 省钱计算器页 | 输入区 | 大号数字输入框，运营商选择胶囊按钮 |
| 省钱计算器页 | 结果区 | 翡翠绿渐变卡片，动态数字跳动动画，策略列表 |
| 我的页面 | 用户信息 | 渐变背景头部，圆形头像，省钱金额高亮 |
| 我的页面 | AI对比 | 表格对比，绿色对勾 vs 红色叉号，动态更新时间戳 |

### 4.3 响应式设计
- 移动优先设计，最大宽度480px居中展示
- 触摸优化：所有可点击区域最小44px，按钮间距≥8px
- 底部导航栏固定，内容区域可滚动
- 横屏时自动切换为双列布局

### 4.4 动效设计
- 雷达扫描：CSS脉冲动画 + 旋转扫描线
- 卡片入场：交错淡入动画（stagger fade-in）
- 数字跳动：省钱金额从0递增到目标值
- 页面切换：底部导航切换时内容区滑入/滑出
- 按钮反馈：点击缩放 + 涟漪效果
